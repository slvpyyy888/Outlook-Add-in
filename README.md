A Outlook Add-In that extend .xml from .pdf.

--how to use--
------------------------------------------------------------------------------------------------------------------------------------------------------------------------
go to ./bin/debug and double click the .vsto file. 
-> download it. 
-> after it open Outlook, on the right, you should see an window with the Add-in. now after selecting the Email (don't open it only click on it) you should see the .xml, .pdf files.
